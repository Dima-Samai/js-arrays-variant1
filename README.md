# Student Management System & Sorting Algorithms Analysis

Цей проєкт виконано в межах Практичної роботи №7 з дисципліни "Основи програмування мовою JavaScript". Програма поєднує інструменти для аналізу масивів даних та практичну реалізацію класичних алгоритмів сортування.

## 🎓 Функціонал системи студентів

* **Управління даними:** Робота з масивом об'єктів (ім'я, вік, GPA, факультет).
* **Динамічна фільтрація:** Пошук за ім'ям та фільтрація за факультетами за допомогою методів `filter()` та `includes()`.
* **Статистичний аналіз:** Автоматичний розрахунок середнього бала (GPA), загальної кількості студентів та виведення ТОП-3 найкращих за допомогою `reduce()` та `sort()`.
* **Інтерактивна таблиця:** Можливість сортування списку за будь-яким полем (ім'я, вік, бал) у прямому та зворотному порядку.

## ⚡ Алгоритми та бенчмарки

У проєкті реалізовано два алгоритми сортування "з нуля" (без використання стандартного `Array.prototype.sort`):

1.  **Bubble Sort (Сортування бульбашкою):** Простий алгоритм з часовою складністю O(n²).
2.  **Quick Sort (Швидке сортування):** Ефективний алгоритм за принципом "розділяй і володарюй" з середньою складністю O(n log n).

### Порівняння продуктивності
Програма містить інструмент для тестування швидкості алгоритмів на масивах різного розміру (100, 1000 та 5000 елементів). Результати виводяться в мілісекундах разом із підрахунком кількості операцій порівняння.

## 🛠 Технічні особливості

* **ES6+ Syntax:** Використання стрілочних функцій, деструктуризації об'єктів, spread-операторів та шаблонних рядків.
* **Методи масивів:** Активне застосування `map`, `filter`, `reduce` для чистоти коду.
* **Performance API:** Використання `performance.now()` для точного вимірювання часу виконання коду.

3. Використовуйте кнопки "Тест елементів", щоб побачити різницю у швидкості алгоритмів у реальному часі.
